Getting Started with Arduino: Blinking LED

Published: 2024-06-01 | Author: VINM Team

Introduction

This tutorial will guide you through your first microcontroller project: making an LED blink using an Arduino board. You'll learn about basic circuit wiring, Arduino code structure, and how to upload your first sketch.

Components Required

  • Arduino Uno board
  • LED
  • 220Ω resistor
  • Breadboard and jumper wires

Circuit Diagram

Code


void setup() {
  pinMode(13, OUTPUT);
}
void loop() {
  digitalWrite(13, HIGH);
  delay(500);
  digitalWrite(13, LOW);
  delay(500);
}

Explanation

The code sets pin 13 as an output and toggles it HIGH and LOW every 500ms, making the LED blink. This is the classic "Hello World" of embedded systems and a great way to get started with Arduino.

Next Steps

  • Try changing the delay to make the LED blink faster or slower.
  • Connect more LEDs to different pins and modify the code to blink them in sequence.
  • Explore using a button to control the LED.
